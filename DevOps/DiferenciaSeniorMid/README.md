# Diferencia Senior vs Mid en DevOps

| Tema        | Mid                | Senior                 |
| ----------- | ------------------ | ---------------------- |
| Pipelines   | Los usa            | Los entiende y corrige |
| Docker      | Corre contenedores | Los construye bien     |
| CI fallando | Espera             | Investiga              |
| Deploy      | Manual             | Automatizado           |
| Infra       | Lejana             | Entendible             |

> Kubernetes, Terraform, Helm, networking complejo son bonus, no requisito base.
