# Herramientas Modernas

## Control de versiones
- Git (rebase, cherry-pick, squash)
- PRs, code review

## Calidad
- Linters
- Formatters
- Tests automatizados

## Observabilidad (conceptual)
- Logs
- Métricas
- Alertas

## Seguridad básica
- Secrets en env vars
- No subir `.env`
- Scans básicos en CI

> Un senior debe saber usar, configurar y explicar estas herramientas.
