# Bulkhead Pattern

Aísla recursos y componentes para evitar que una falla afecte a todo el sistema. Divide el sistema en compartimentos independientes.

**Ventajas:**
- Mejora la resiliencia.
- Limita el impacto de fallos.

**Trade-off:**
- Puede aumentar la complejidad.
- Requiere diseño cuidadoso de límites.
