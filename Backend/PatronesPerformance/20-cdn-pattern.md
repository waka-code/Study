# Content Delivery Network (CDN) Pattern

Distribuye contenido desde servidores cercanos al usuario para reducir latencia y mejorar la velocidad de carga. Usado para archivos estáticos, imágenes, videos.

**Ventajas:**
- Reduce la latencia.
- Mejora la experiencia global.

**Trade-off:**
- Puede aumentar el costo.
- Requiere gestión de caché y actualización.
