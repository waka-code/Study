# Lazy Loading Pattern

Carga recursos o datos solo cuando realmente se necesitan, en vez de al inicio. Evita cargar información innecesaria, ahorrando memoria y tiempo.

**Ventajas:**
- Reduce el uso inicial de recursos.
- Mejora el tiempo de arranque.

**Trade-off:**
- Puede causar retrasos cuando se accede por primera vez.
- Complejidad en la gestión de dependencias.
