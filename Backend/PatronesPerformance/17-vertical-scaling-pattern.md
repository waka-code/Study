# Vertical Scaling Pattern

Mejora el rendimiento aumentando los recursos (CPU, RAM) de una sola instancia. Útil para cargas intensivas en un solo servidor.

**Ventajas:**
- Fácil de implementar.
- Mejora el rendimiento rápidamente.

**Trade-off:**
- Límite físico de escalabilidad.
- Puede ser costoso y no tolerante a fallos.
