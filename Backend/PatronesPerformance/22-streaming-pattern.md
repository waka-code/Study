# Streaming Pattern

Procesa o envía datos en flujo continuo, sin cargar todo en memoria. Usado para archivos grandes, video, audio, o datos en tiempo real.

**Ventajas:**
- Reduce el uso de memoria.
- Permite procesamiento en tiempo real.

**Trade-off:**
- Complejidad en el manejo de errores.
- Requiere gestión de buffers y sincronización.
