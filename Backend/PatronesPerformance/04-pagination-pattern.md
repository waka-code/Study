# Pagination Pattern

Divide grandes conjuntos de datos en páginas, mostrando solo una parte a la vez. Reduce la carga en el servidor y el consumo de memoria.

**Ventajas:**
- Mejora la experiencia de usuario.
- Reduce el uso de recursos.

**Trade-off:**
- Complejidad en la gestión de navegación.
- Puede requerir lógica adicional para búsquedas y filtros.
