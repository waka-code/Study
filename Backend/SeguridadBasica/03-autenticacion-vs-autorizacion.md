# Autenticación vs Autorización

## Autenticación
Proceso de verificar la identidad de un usuario (¿quién eres?). Ejemplo: login con usuario y contraseña.

## Autorización
Proceso de verificar si el usuario tiene permisos para realizar una acción específica (¿qué puedes hacer?). Ejemplo: acceso a recursos restringidos según rol.

**Diferencia clave:**
- Autenticación: acceso al sistema.
- Autorización: acceso a funcionalidades o datos dentro del sistema.
