# Prefetching Pattern

Carga anticipada de datos que probablemente serán requeridos próximamente, mejorando la experiencia del usuario y reduciendo la latencia.

**Ventajas:**
- Reduce el tiempo de espera.
- Mejora la fluidez de la aplicación.

**Trade-off:**
- Puede cargar datos innecesarios.
- Aumenta el consumo de recursos.
