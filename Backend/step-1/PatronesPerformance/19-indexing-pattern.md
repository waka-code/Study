# Indexing Pattern

Utiliza índices en bases de datos para acelerar búsquedas y consultas. Fundamental para optimizar el acceso a grandes volúmenes de datos.

**Ventajas:**
- Mejora la velocidad de consulta.
- Reduce el tiempo de respuesta.

**Trade-off:**
- Aumenta el uso de almacenamiento.
- Puede ralentizar operaciones de escritura.
