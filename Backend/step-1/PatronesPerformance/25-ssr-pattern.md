# Server-Side Rendering (SSR) Pattern

Renderiza el contenido en el servidor antes de enviarlo al cliente, mejorando el tiempo de carga inicial y el SEO. Usado en aplicaciones web modernas.

**Ventajas:**
- Mejora el tiempo de carga inicial.
- Optimiza el SEO.
