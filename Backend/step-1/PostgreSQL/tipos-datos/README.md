# 2️⃣ Tipos de datos avanzados (VENTAJA DE POSTGRES)

Aquí PostgreSQL **gana por goleada**.

## Debes conocer y saber cuándo usar
- `UUID`
- `JSON` vs `JSONB`
- `ARRAY`
- `ENUM`
- `TIMESTAMP WITH TIME ZONE`
- `INET`, `CIDR`
- `NUMERIC` vs `FLOAT`
- Tipos custom

> ¿Por qué `JSONB` es mejor que `JSON`? Porque es binario, indexable y más rápido para queries.
