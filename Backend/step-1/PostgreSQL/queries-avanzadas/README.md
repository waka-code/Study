# 5️⃣ Queries avanzadas (SQL real)

No solo CRUD.

## Debes dominar
- `JOIN` (INNER, LEFT, RIGHT)
- Subqueries
- CTE (`WITH`)
- Window functions: `ROW_NUMBER`, `RANK`, `OVER (PARTITION BY)`
- `GROUP BY` / `HAVING`
- `UNION` / `UNION ALL`
- `EXISTS` vs `IN`

> **Window functions separan mid de senior.**
