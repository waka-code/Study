# npm / yarn

npm y yarn son gestores de paquetes para JavaScript. Permiten instalar, actualizar y gestionar dependencias de proyectos.

**Ventajas:**
- Instalación rápida de librerías.
- Scripts de build, test y deploy.
- Gestión de versiones y lockfiles.

**Uso básico:**
- `npm install paquete` o `yarn add paquete`.
- Scripts en `package.json`.
