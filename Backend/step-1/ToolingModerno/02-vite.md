# Vite

Vite es una herramienta de build y desarrollo rápido para proyectos frontend. Utiliza ES Modules y un servidor de desarrollo ultrarrápido.

**Ventajas:**
- Hot Module Replacement instantáneo.
- Configuración mínima.
- Soporte para React, Vue, Svelte, etc.

**Uso básico:**
- Configuración en `vite.config.js`.
- Ejecuta `npm run dev` para desarrollo.
