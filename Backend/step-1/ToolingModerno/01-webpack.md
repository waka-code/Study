# Webpack

Webpack es un empaquetador de módulos para aplicaciones JavaScript modernas. Permite combinar, transformar y optimizar archivos (JS, CSS, imágenes) en bundles para producción.

**Ventajas:**
- Control total sobre el proceso de build.
- Plugins y loaders para transformar recursos.
- Code splitting y lazy loading.

**Uso básico:**
- Configuración en `webpack.config.js`.
- Ejecuta `npx webpack` para generar el bundle.
