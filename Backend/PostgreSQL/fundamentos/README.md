# 1️⃣ Fundamentos sólidos (pero bien entendidos)

No es opcional, pero **a nivel profundo**.

## Debes dominar
- Tablas, filas, columnas
- Tipos de datos nativos
- Constraints: `PRIMARY KEY`, `FOREIGN KEY`, `UNIQUE`, `CHECK`, `NOT NULL`

> **Las constraints no son opcionales**, son parte de la lógica del sistema.
