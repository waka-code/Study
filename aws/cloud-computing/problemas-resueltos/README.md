# Problemas resueltos por el Cloud Computing

## 1. Flexibilidad
La nube permite ajustar recursos y servicios según las necesidades del negocio sin depender de hardware físico.

**Ejemplo:** Cambiar la configuración de una base de datos en minutos.

---

## 2. Rentabilidad
El cloud elimina la compra de servidores costosos y reduce mantenimiento, electricidad y soporte. Pagas solo por lo que utilizas.

**Ejemplo:** Una pyme paga solo por el almacenamiento que usa cada mes.

---

## 3. Escalabilidad
La nube permite aumentar o disminuir recursos (CPU, RAM, almacenamiento, instancias, etc.) sin interrupciones.

**Ejemplo:** Un sitio web escala automáticamente durante un evento viral.

---

## 4. Elasticidad
La nube ajusta automáticamente los recursos en tiempo real según la demanda.

**Ejemplo:** Un servicio de streaming escala servidores durante un estreno popular.

---

## 5. Alta disponibilidad y tolerancia a fallos
Los proveedores de nube diseñan sus servicios para resistir fallos de hardware, red o zona geográfica.

**Ejemplo:** Una app sigue funcionando aunque falle un servidor o una zona completa.

---

## 6. Agilidad
Los equipos pueden crear entornos en minutos, probar ideas rápidamente y desplegar más rápido.

**Ejemplo:** Un equipo de desarrollo lanza una nueva funcionalidad en horas, no semanas.
