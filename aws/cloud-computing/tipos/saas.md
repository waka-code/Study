# SaaS – Software como Servicio

El proveedor ofrece la **aplicación completa lista para usar**, accesible por web o app. No manejas nada de infraestructura ni despliegues.

## El proveedor administra TODO
- Servidores
- SO
- App
- Seguridad
- Actualizaciones
- Disponibilidad

## Ejemplos
- Gmail
- Office 365
- Slack
- Salesforce
- Shopify

## Ideal cuando
Solo quieres usar la herramienta sin preocuparte por instalación o mantenimiento.
