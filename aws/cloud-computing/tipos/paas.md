# PaaS – Plataforma como Servicio

El proveedor ofrece una **plataforma lista** para desplegar aplicaciones sin preocuparte por servidores, parches o SO. Tu foco es el código y la lógica del negocio.

## El proveedor administra
- Infraestructura
- Sistema operativo
- Middleware
- Entorno de ejecución

## Tú administras
- Código
- Datos

## Ejemplos
- Heroku
- Firebase
- AWS Elastic Beanstalk
- Vercel (para apps web)
- Azure App Service

## Ideal cuando
Quieres desarrollar y desplegar rápido sin manejar infraestructura.
