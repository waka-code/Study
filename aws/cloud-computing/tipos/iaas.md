# IaaS – Infraestructura como Servicio

La nube te entrega **infraestructura básica**: servidores, redes, almacenamiento, máquinas virtuales y firewalls. Tú administras el sistema operativo, tus aplicaciones y la configuración.

## Lo que te da el proveedor
- Servidores virtuales (VMs)
- Redes (VPC, balanceadores de carga)
- Almacenamiento (bloque, archivo, objeto)
- Seguridad a nivel infraestructura

## Tú administras
- Sistema operativo
- Aplicaciones
- Middleware
- Datos

## Ejemplos
- AWS EC2
- Azure VMs
- Google Compute Engine
- DigitalOcean Droplets

## Ideal cuando
Necesitas control total, arquitectura personalizada o migrar sistemas legacy.
