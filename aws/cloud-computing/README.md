# Cloud Computing

Cloud computing es la entrega de servicios de computación (servidores, almacenamiento, bases de datos, redes, software, análisis, inteligencia artificial, etc.) a través de Internet ("la nube"). Permite acceso flexible, escalable y bajo demanda a recursos tecnológicos.

## Modelos principales
- Cloud privado
- Cloud público
- Cloud híbrido

## Subcarpetas
- `modelos/`: Definiciones de cloud privado, público e híbrido.
- `caracteristicas/`: Explicación de las características esenciales del cloud.
- `ventajas/`: Ventajas clave del cloud computing.
- `problemas-resueltos/`: Problemas que resuelve el cloud.
