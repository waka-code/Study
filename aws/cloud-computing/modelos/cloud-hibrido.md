# Cloud Híbrido

El **cloud híbrido** combina un cloud privado con un cloud público, permitiendo mover datos y aplicaciones entre ambos según convenga.

## Características clave
- Flexibilidad para usar lo mejor de ambos mundos.
- Información sensible en el cloud privado y cargas variables en el cloud público.
- Ideal cuando necesitas control y seguridad, pero sin renunciar a escalabilidad.

## Ejemplo
Una empresa de salud almacena historiales médicos en un cloud privado y usa el cloud público para procesar grandes volúmenes de datos durante campañas de salud.
