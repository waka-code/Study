# Servicio medido (Measured Service)

El uso de los recursos se mide constantemente (CPU, ancho de banda, almacenamiento, instancias activas) y el cliente paga solo por lo que consume.

## Ejemplo
Una base de datos en la nube consume 20 GB de almacenamiento en enero y 50 GB en febrero; la factura refleja solo el uso real de cada mes.
