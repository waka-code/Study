# Alquiler múltiple y agrupación de recursos (Resource Pooling / Multitenancy)

Los recursos físicos (CPU, RAM, almacenamiento, red) se comparten entre múltiples clientes, pero cada uno aislado y seguro. El proveedor de la nube agrupa recursos y los asigna dinámicamente según demanda.

## Ejemplo
AWS ejecuta máquinas virtuales de diferentes empresas en los mismos servidores físicos, pero cada una está virtualmente separada y segura.
