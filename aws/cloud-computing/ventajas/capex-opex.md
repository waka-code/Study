# Cambia el gasto capital (CAPEX) por gasto operativo (OPEX)

En lugar de invertir grandes cantidades de dinero en comprar servidores, redes o centros de datos (CAPEX), las empresas pagan solo por lo que usan cada mes (OPEX).

## Ejemplo
Una empresa deja de comprar servidores y paga mensualmente por el uso de AWS EC2 y S3.
