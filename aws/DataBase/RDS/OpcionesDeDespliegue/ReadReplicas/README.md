# Read Replicas

## Descripción
- Réplicas de solo lectura que se pueden implementar en la misma región o en regiones diferentes.
- Ayudan a escalar la carga de lectura.

## Beneficios
- Escalabilidad horizontal para lecturas intensivas.
- Réplicas en múltiples regiones para mejorar la latencia global.

## Casos de uso
- Aplicaciones con alta carga de lectura.
- Escenarios de análisis y reporting.
