# Global Databases

## Descripción
- Réplicas en múltiples regiones para recuperación ante desastres y baja latencia global.
- Disponible para Amazon Aurora.

## Beneficios
- Recuperación ante desastres en menos de un minuto.
- Baja latencia para usuarios globales.

## Casos de uso
- Aplicaciones globales.
- Continuidad del negocio y recuperación ante desastres.
