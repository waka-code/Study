# Multi-AZ Deployment

## Descripción
- La base de datos principal se replica automáticamente en una AZ secundaria.
- AWS gestiona el failover automático en caso de fallo.

## Beneficios
- Alta disponibilidad.
- Failover automático.
- Ideal para aplicaciones críticas.

## Casos de uso
- Aplicaciones empresariales.
- Workloads de producción.
