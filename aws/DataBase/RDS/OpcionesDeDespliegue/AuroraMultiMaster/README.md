# Aurora Multi-Master

## Descripción
- Disponible solo para Amazon Aurora.
- Permite múltiples instancias de escritura en diferentes AZs.

## Beneficios
- Alta disponibilidad para escrituras.
- Reducción de latencia en escrituras distribuidas.

## Casos de uso
- Aplicaciones globales con escrituras concurrentes.
- Workloads de misión crítica.
