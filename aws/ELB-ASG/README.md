# ELB y ASG

En esta carpeta encontrarás información sobre dos servicios clave de AWS:

1. **Elastic Load Balancing (ELB):**
   - Distribuye tráfico entre múltiples recursos.
   - Garantiza alta disponibilidad y escalabilidad.

2. **Auto Scaling Group (ASG):**
   - Escala automáticamente las instancias EC2.
   - Mantiene la capacidad adecuada según la carga.

Ambos servicios trabajan juntos para garantizar que tus aplicaciones sean altamente disponibles, escalables y tolerantes a fallos.
