# Amazon FSx

**Amazon FSx** es un servicio de **sistemas de archivos totalmente administrados** para **workloads específicos** que requieren **alto rendimiento** o **compatibilidad nativa** con ciertos sistemas de archivos.

## Tipos de Amazon FSx

### FSx for Windows File Server

- Basado en **SMB**
- Integración con **Active Directory**
- Soporta ACLs de Windows
- Alta disponibilidad

### FSx for Lustre

- Muy baja latencia
- Altísimo throughput
- Integración con **Amazon S3**

### FSx for NetApp ONTAP

- NFS, SMB, iSCSI
- Snapshots avanzados
- Replicación
- Data tiering

### FSx for OpenZFS

- Baja latencia
- Snapshots rápidos
- Compresión
- Clones
