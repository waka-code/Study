# Objeto

Un objeto en Amazon S3 es la unidad básica de almacenamiento. Cada objeto consta de datos, metadatos y una clave única que lo identifica dentro de un bucket.

## Componentes
- **Datos:**
  - El contenido del archivo.
- **Metadatos:**
  - Información adicional como tipo de contenido, fecha de creación, etc.
- **Clave:**
  - Identificador único del objeto dentro del bucket.

## Ejemplo
- Nombre del objeto: `documento.pdf`
- Metadatos: `Content-Type: application/pdf`
- Clave: `archivos/documento.pdf`
