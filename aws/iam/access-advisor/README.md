# IAM Access Advisor

Muestra los permisos de servicio concedidos a un usuario y cuándo se accedió a esos servicios por última vez.

## ¿Para qué sirve?
- Revisar y ajustar políticas de acceso.
- Identificar permisos no utilizados.

**Cómo acceder:**
1. Ve a la consola de IAM > Usuarios > [usuario] > Access Advisor.
2. Consulta los servicios y la última fecha de acceso.
