# CloudShell en AWS

CloudShell es una terminal administrada por AWS para ejecutar comandos CLI sin configurar credenciales locales.

- Funciona a nivel de región.
- Permite acceso rápido y seguro a recursos AWS desde el navegador.
- Incluye herramientas preinstaladas (AWS CLI, Python, Node.js, etc.).

**Ejemplo de uso:**
1. Abre la consola de AWS y selecciona CloudShell (ícono de terminal).
2. Ejecuta comandos como `aws s3 ls` directamente desde el navegador.
