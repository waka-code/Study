# ¿Qué es React?

React es una biblioteca de JavaScript para construir interfaces de usuario de forma declarativa, basada en componentes reutilizables.

- Declarativo: describes el UI y React se encarga del DOM.
- Basado en componentes: piezas reutilizables y aisladas.
- Virtual DOM: actualizaciones eficientes.

```jsx
const element = <h1>Hello, world!</h1>;
```
