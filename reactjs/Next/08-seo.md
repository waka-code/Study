# SEO técnico en Next.js

Next.js ofrece Metadata API, SEO dinámico en server, OpenGraph, sitemap, robots, canonical URLs.

**Ejemplo:**
```js
export const metadata = {
  title: 'Página',
  description: 'Descripción',
  openGraph: { title: 'OG Title' }
};
```

**Nota:** SSR impacta positivamente en SEO.
